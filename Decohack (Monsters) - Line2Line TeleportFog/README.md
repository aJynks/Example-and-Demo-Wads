# Line2Line Teleport Monster Hack to trigger FOG on port

This is some DECOHack code showing how to build a deep copy of a Doom monster so it will spawn a teleport fog when it first sees the player. This enables you to use silent line-to-line teleports, which support z-height, among other things while still having teleport fogs appear as they arrive.

In the demo I have built a Cacodemon and a Hell Knight version to show flying and ground-based line-to-line teleporting.
