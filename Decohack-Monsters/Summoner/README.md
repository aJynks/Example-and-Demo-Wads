This project contains three versions of the Summoner.

- My Original Version
  summoner_v1.wad

  This version of the monster uses the original GFX
  taken from Realm667. It uses a spawn box as well as
  a green TeleportFog. This version can also take
  radius damage.


- numOsic's Version
  summoner_v2_numOsic.wad

  This version tweaks how the fire trail looks.
  A "volley fire" mode has been added when the player
  gets close, and the spawn box and fog have been
  replaced with a red flame spawner graphic.
  It is immune to radius damage, like the Cyberdemon.


- My Edit of numOsic's Version (Closer to my original
  design)
  summoner_v3.wad

  This version is a mix of the two previous versions.
  It mainly uses numOsic's superior code, but the
  volley fire mode has been removed and the original
  spawn box and TeleportFog have been restored.

  In addition, a new green flame has been added at the
  spawn point so it synchronises better with the
  teleport fog. The sprite graphics have also been
  slightly recoloured to a more intense blue.

  This version is also immune to radius damage.
